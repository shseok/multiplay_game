# mp_game

A new Flutter project.

## Getting Started

## directory
1. widgets - 재사용 가능 위젯 (ex. 참가or생성 버튼)
2. utils - 재사용 색상, 함수
3. responsive - 웹을 고려

4. server
    - 유저가 방을 만듦 -> 데이터 베이스에 저장(다른 유저 참가 가능)

5. socket.io
   - socket_io_client와 socket.io의 버전을 맞춰야한다.


# Info
- promise in JS = Future in dart
- socket in node = stream in flutter
